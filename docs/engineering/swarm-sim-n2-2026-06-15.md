# Deterministic Swarm Sim N2 2026-06-15

## Thesis

A deterministic N=2 grid swarm can demonstrate the accountable-swarm loop
before any physical hardware: local collision guards choose `MOVE`, `REROUTE`,
or `HOLD`, each agent emits a replayable DecisionTrace, and the aggregate
report proves no same-cell or swap collision occurred.

## Why This Matters

The project needs a swarm-shaped artifact before SO-101 hardware or a DimOS
physics scene is available. This gate tests the trace spine across multiple
local actors while keeping Qwen out of the real-time loop.

## Smallest Falsifying Experiment

Run two agents through a corridor-crossing scenario where greedy direct paths
would conflict. A deterministic local guard must choose a reroute, both agents
must reach goals, and both traces must replay.

## Commands

```bash
python3 scripts/run_swarm_sim.py \
  --agents 2 \
  --ticks 8 \
  --trace-dir runs/swarm/n2 \
  --report-out runs/swarm/n2_report.json
```

## GO Gate

- `scripts/run_swarm_sim.py --agents 2` exits `0`.
- Every emitted trace passes `scripts/verify_trace.py`.
- `replay_swarm_traces()` reconstructs final positions and collision counts
  from the trace artifacts without using the original simulation result object.
- The report records zero same-cell collisions.
- The report records zero swap collisions.
- The report and traces contain no raw floats because trace canonical JSON
  rejects raw float values.
- `./scripts/local_gate.sh` passes.

## Qodo Review Remediation

Qodo flagged two hardening points on PR #12:

- `GO` reports must require one verified trace summary per configured agent.
- Hash verification alone is not enough replay evidence for swarm decisions.

The implementation now requires exact trace-summary coverage for all configured
agent IDs and exposes `replay_swarm_traces()`, which reconstructs final
positions plus same-cell and swap collision counts from the DecisionTrace
artifacts.

## Non-Claims

- No physical robot behavior.
- No SO-101 operation.
- No real-time latency or reliability claim.
- No 3D physics simulation.
- No DimOS integration.
- No Alibaba deployment proof.
- No claim that larger or learned swarms succeed.
